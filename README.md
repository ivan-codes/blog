# Blog

A website to host future blog posts I may or may not write.

### 👩‍💻 Setup

1. `npm run dev` or `yarn dev`
2. Visit http://localhost:3000

### ⚒ Technologies

- Next.js
- Typescript
- Tailwind CSS

### ✨ Features

- Markdown to HTML
- Light and dark mode
- Highlight.js syntax highlighting
- Next.js static generation
